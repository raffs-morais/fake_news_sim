Pré-requisitos para rodar o projeto:
Python 3.8 ou superior

Instalação:
(Opcional mas recomendado) Cire um ambiente virtual --> python -m venv venv
     Ative o ambiente virtual:
         Windows --> venv\Scripts\activate
         Linux/MacOS --> source venv/bin/activate

Instale as dependências:
pip install -r requirements.txt

Como rodar: O projeto tem dois modos de uso independentes. Escolha o que preferir.

     Modo 1 — Interface Web (Streamlit), Recomendado:
          streamlit run app.py
          O navegador abrirá automaticamente em http://localhost:8501.

     Modo 2 — Terminal Interativo:
          Rode o main.py e sera exibido um menu com as opções disponíveis.
